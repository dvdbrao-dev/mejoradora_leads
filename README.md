# Mejoradora Leads

Consultoría energética de David Braojos. Herramienta interna de captación y
gestión de leads para cambio de comercializadora y PPA solar Soldelia.

## Quick start

```bash
cd ~/mejoradora_leads
source .venv/bin/activate
source ~/.env.mejoradora
make help
```

Contexto permanente:

```bash
./bin/brief
make context
```

Operación manual actual:

```bash
make review-csv
make dashboard
```
